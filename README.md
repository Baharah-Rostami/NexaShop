# NEXASHOP - Product Store App

## Project Description

NEXASHOP is a modern product store application built with React.
Users can browse products, view product details, manage a shopping cart, and customize app settings like theme.

This project demonstrates the use of Context API, Redux Toolkit, and React Query in a real-world application.

---

##  Features

* 🛒 Add, remove, increase, and decrease cart items
* 💰 Display total items and total price
* 🔍 Search products by title
* ❤️ Add/remove favorites
* 🌙 Dark mode / Light mode toggle
* 📦 Product details page
* ⚡ Skeleton loading for better UX
* 💾 Cart persistence using localStorage
* 🔐 Simple authentication (signup/login with localStorage)

---

##  Tools & Libraries Used

* React
* React Router DOM
* Redux Toolkit
* React Query (@tanstack/react-query)
* Context API + useReducer
* Tailwind CSS
* React Hook Form
* Lucide React (icons)

## 🚀 How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/Baharah-Rostami/nexashop.git
```

2. Navigate to the project folder:

```bash
cd nexashop
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

5. Open in browser:

```bash
http://localhost:5173
```
